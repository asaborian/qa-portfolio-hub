## 💬 Interview FAQ

### Q: なぜ38歳実務未経験からITエンジニアを目指すに至ったのか？
A: もともと仕事でExcelマクロを使うようになり、プログラミングが楽しいと感じて仕事にしたいと考えたのが始まりです。当初は、せっかくなら組織に属さず個人開発者として独立する夢を見ながら、趣味として独学を続けていました。しかし、子どもが生まれたことをきっかけに、今後の生活設計を夫婦で話し合った結果、現実的な選択として「リモートワークができる企業への転職」が最適であるという結論に至りました。そこで、自分のスキルや興味を活かせる道を模索した結果、セキュリティに強いQAエンジニアというポジションであれば、この年齢から独学で挑戦しても、現在の市場の需要と供給のバランスから見て十分に採用していただける可能性があると考えました。自分の興味とも完全に一致しているため、このタイミングでの転職を決意しました。

### Q: ITエンジニアを目指す理由は？
A: プログラミングによる業務効率化が楽しいからです。手作業だった仕事を自分の書いたコードで自動化できたことにやりがいを感じ、これを本業にしたいと考えました。

### Q: QAエンジニアを目指す理由(内面的理由)は？
A: QAエンジニアの業務は、テストの自動化や、セキュリティの観点から設計に関わるなど、同僚の業務負担を減らすことにおいて長期的にレバレッジが効く業務であるため、やりがいを感じられると考えたからです。

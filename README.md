- **2026-07-01 23:04**: json-serverの挙動 / GET: 全件または特定レコードを返す / POST: 新規作成されたレコードを返す / PUT: 更新されたレコードを返す / DELETE: 空オブジェクト{}を返す

## 💬 Interview FAQ

### Q: 就職したらどのように貢献できるか？
A: セキュリティスペシャリスト(SC)の知識を活かし、JSTQBの標準的なテスト技法に基づいて、Playwrightを活用したテスト自動化を構築できます。

### Q: なぜ38歳実務未経験からITエンジニアを目指すに至ったのか？
A: 現職の業務効率化のためにExcelマクロを使うようになり、プログラミングが楽しいと感じて仕事にしたいと考えたのが始まりです。当初は、個人開発と案件の両軸で生計を立てるフリーランスエンジニアを目指して独学を続けていました。しかし、子どもが生まれたことをきっかけに、今後の生活設計を夫婦で話し合った結果、現実的な選択として「リモートワークができる企業への転職」が最適であるという結論に至りました。そこで、自分のスキルや興味を活かせる道を模索した結果、セキュリティに強いQAエンジニアというポジションであれば、この年齢から独学で挑戦しても、現在の市場の需要と供給のバランスから見て十分に採用していただける可能性があると考えました。自分の興味とも完全に一致しているため、このタイミングでの転職を決意しました。

### Q: 定期的な出社は可能か？ 
A: 子どもの幼稚園の送り迎えがあるため、原則リモートを希望しますが、月１回程度の頻度や、事前に調整ができるスケジュールであれば、家族と調整のうえ出社可能です。
